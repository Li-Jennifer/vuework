# Vue 实战项目硅谷外卖

1) 此项目为外卖 Web App (SPA) 

2) 包括商家, 商品, 购物车, 用户等多个子模块 

3) 使用 Vue 全家桶+ES6+Webpack 等前端最新最热的技术 

4) 采用模块化、组件化、工程化的模式开发

## 1. 准备

	1) 确保安装了node环境
		查看是否已经安装: node -v
	2) 确保安装了mongodb, 并启动了对应的服务
		查看是否安装并启动了服务: 右键-->任务管理器-->服务-->MongoDB

## 2. 启动后台应用

	1). 进入gshop-server
	2). 执行命令: npm start

## 3. 启动前台应用并访问

	1). 进入gshop-client
	2). 执行命令: npm start
